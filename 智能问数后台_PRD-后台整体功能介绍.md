## 2. 产品概述

### 2.1 产品定位

> 智能问数后台系统是**智能问数系统的独立配置端**，用于配置分析方法论、报告模型、管理权限和审计日志。配置信息实时同步至智能问数系统前台，为AI分析提供标准化的方法论框架和报告输出格式。

### 2.2 核心价值

- **方法论配置**：定义和管理数据分析的方法论框架，供AI挖掘助手调用执行标准化分析流程
- **报告模型配置**：配置报告模板，定义叙事结构、章节排版、图表规则、视觉样式
- **权限管理**：管理系统组织架构、角色权限和人员账号
- **日志审计**：记录系统操作日志，支持查询和导出

## 4. 整体业务流程

### 流程1：分析方法论配置

```
`开始 → 分析简介填写 → 核心分析逻辑 → 整体输入说明 → 整体输出说明 →`
`步骤拆解 → 步骤的描述 → 步骤的输入 → 步骤的输出 → 完成
```

### **流程2：报告模型配置**

- **流程2.1：内置配置-叙事结构配置**

  ```
  开始 → 叙事结构配置 → 叙事的步骤 → 叙事的描述 → 完成
  ```

- 流程2.2：内置配置-章节排版配置

  ```
  开始 → 描述排版模式 → 完成
  ```

- 流程2.3：内置配置-颜色主题配置

  ```
  开始 → 描述颜色风格 → 选择颜色 → 完成
  ```

- 流程2.4：报告模板配置

  ```
  开始 → 模板简介 → 选择叙事结构 → 选择章节排版 → 选择图表使用 →
  选择主题颜色 → 配置建议与风险 → 配置业务执行 → 完成
  ```

## 11. 异常流程与边界条件说明

### 11.1 全局通用规则

| 类别        | 场景/条件             | 处理规则                                       |
| :-------- | :---------------- | :----------------------------------------- |
| **网络异常**  | 请求超时、无网络连接        | 前端展示统一提示"网络异常，请稍后重试" |
| **权限不足**  | 用户访问无权限页面或执行无权限操作 | 前端跳转至403提示页或弹窗提示"您暂无此操作权限" |
| **数据为空**  | 列表、下拉选项无数据        | 展示统一的空状态插画与提示文案 |
| **并发冲突**  | 同一数据被多人同时修改       | 后提交者操作失败，提示"数据已被他人修改，请刷新后重试" |
| **会话过期**  | 登录状态过期           | 自动跳转登录页，提示"登录已过期，请重新登录" |

### 11.2 模块特定规则

| 模块       | 特定异常场景  | 处理规则简述                               |
| :------- | :------ | :----------------------------------- |
| **分析方法论** | 分析名称重复 | 提示"该分析名称已存在，请更换" |
| **报告模板** | 模板名称重复 | 提示"该模板名称已存在，请更换" |
| **权限管理** | 工号重复 | 提示"工号已存在" |
| **权限管理** | 手机号重复 | 提示"手机号已存在" |
| **权限管理** | 删除有子部门的部门 | 提示"该部门存在子部门，无法删除" |
| **权限管理** | 删除有人员的部门 | 提示"该部门存在人员，无法删除" |
