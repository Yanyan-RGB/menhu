## 2. 产品概述

### 2.1 产品定位

> 智能数据分析系统是一款**AI驱动的企业级数据决策平台**，通过整合多源数据连接、智能建模、指标管理、知识库、AI分析、业务监控、问题定位、改进分析，实现从数据接入到智能决策的完整闭环。
>
> 其中指标建立接入ai语义创建，数据模型加入专业知识模型+ai语义创建，分析流程结合专业分析步骤+ai语义分析

### 2.2 核心价值

#### 2.2.1 分析功能

- **数据整合**：统一接入企业内外部多源数据
  - 数据接入：API、数据库、表格导入
- **数据建模**：自动建模、表与不同维度的关系查询
  - 表与表之间的关系
  - 表与字段之间的关系
  - 表与指标之间的关系
  - 表与数据模型之间的关系
  - 表与挖掘模型之间的关系
  - 字段与字段之间的关系
  - 指标与指标之间的关系
  - 数据模型与挖掘模型之间的关系
- **挖掘分析**：建立模型，支持AI自动发现数据洞察，降低分析门槛
  - **指标建立**：AI生成指标口径、支持基础/派生/复合指标AI创建与校验
  - **维度管理**：建立字段维度和标签维度管理
  - **数据模型**：结合业务销售增长/客户生命周期/价值分析模型，建立观测指标
  - **分析方法**：结合归因分析/帕累托分析/波士顿矩阵/销售预测分析等，建立分析步骤
  - **报告模型**：建立报告模式，叙事结构（分层/对比/递进/总分）、排布方式、图表规则、视觉、行动建议、业务员执行配置报告风格
  - **挖掘配置**：将指标、数据模型、分析方法、报告模型多样式组合

#### 2.2.2 业务功能

- **智能问数**：AI数据报告、AI智能问数、AI通用问答
  - AI智能报告：分析报告、洞察与建议设置监控、业务执行推送执行数据
  - AI智能问数：快速分析指标
  - AI通用问答：指标口径、简单取数报表
- **分析推送**：对报告分享
  - 针对分析报告可以常规订阅，固定查看
  - 固定报告可以在系统内分享给其他人，相关监控等信息一并分享
- **知识库**：业务知识库、行业知识库
- **使用指引**：系统功能引导和使用说明

#### 2.2.3 业务价值

- **决策支持**：生成可执行的分析报告和建议
- **业务闭环**：支持从分析到执行的全流程管理
- **影响分析**：分析决策的效果影响，权重占比

### 6.2 详细业务流程

#### 阶段一：基础配置初始化

- **流程1：数据接入流程**
  ```
  开始 → 选择数据源类型 → 配置连接参数 → 测试连接 → 
  选择数据表 → 创建数据集 → 完成
  ```
- **流程2：数据建模流程**
  ```
  开始 → 选择数据集 → 配置实体关系 → 定义表间关联 → 
  配置维度属性 → 验证模型 → 保存 → 完成
  ```
- **流程3：指标定义流程**
  ```
  开始 → 选择指标类型(基础/派生/复合) → AI填写基础信息 → 
  配置计算逻辑 → 关联维度 → 提交验证 → 
  验证通过 → 发布 → 完成
  ```

#### 阶段二：深度分析配置

- **流程4：数据模型配置**
  
  ```
  开始 → 模型简介填写 → 选择此模型的相关指标 → 指标分类 → 保存/发布 → 完成
  ```
  
- **流程5：分析方法论配置**
  
  ```
  开始 → 分析简介填写 → 核心分析逻辑 → 整体输入说明 → 整体输出说明 →
  步骤拆解 → 步骤的描述 → 步骤的输入 → 步骤的输出 → 完成
  ```
  
- **流程6：挖掘配置流程**
  
  ```
  开始 → 创建助手 → 配置基础信息 → 绑定数据模型 → 
  绑定方法论 → 选择报告模板 → 配置洞察规则 → 保存版本 → 
  发布 → 完成
  ```

#### 阶段三：日常使用（前台应用）

- **流程7：智能分析流程**
  
  ```
  开始 → 进入智能决策 → 选择挖掘助手 → 
  选择分析模式(报告/问答/通用) → 输入分析需求 → 
  AI理解需求 → 生成报告大纲 → 生成完整报告 → 
  查看洞察 → 订阅/导出 → 监控 → 完成
  ```
  
- **流程8：报告分享与推送**
  
  ```
  开始 → 选择分享人员 → 选择推送渠道 → 点击查看报告 → 完成
  ```

#### 阶段四：决策执行

- **流程10：业务执行流程**

  ```
  开始 → 查看分析报告 → 查看洞察建议 → 选择业务动作 → 
  配置动作参数 → 确认执行 → 触发业务系统 → 
  追踪执行状态 → 查看执行跟进分析 → 完成
  ```

## 7. 涉及AI使用流程说明

- 指标管理-创建基础指标
  ```
  开始→用户输入基础指标创建话术→AI 语义解析（拆解基础指标名称、统计口径、聚合方式、关联数据源、筛选条件）→元数据校验（基础指标相关字段合法性、关联数据表校验）→平台已配置基础指标优先检索复用→无复用则进行同名基础指标存在校验→存在同名基础指标选择覆盖 / 另存→前端回填基础指标标准化口径、关联数据表及关联字段→用户确认修改→存入基础指标库（绑定关联原子指标及血缘关系）→生成标准 MQL 语句→MQL 引擎结合指标血缘 + 后台已定基础指标公式编译底层 SQL→业务场景调用执行→结束
  ```
  
- 智能分析-AI数据报告
  ```
  开始 → 用户选择挖掘助手并输入问题 → 系统解析问题提取时间范围和意图 →
  
  若意图模糊或存在歧义 → 系统生成澄清问题 → 用户明确选择 → 返回主流程
  
  若所需指标不存在 → 系统推断指标逻辑并生成文字描述 → 用户确认 → 自动保存为基础指标 → 返回主流程
  
  若意图清晰且指标完备 → 调用数据模型生成查询 → 结合表血缘补全关联 → 执行查询获取数据 → 调用对应分析模型 → 调用报告模型渲染排版 → 生成报告返回用户 → 结束
  ```
  
- 智能分析-AI智能问数
  ```
  开始 → 用户输入问题 → 系统解析提取查询意图和条件 →
  
  若意图模糊 → 系统生成澄清问题 → 用户明确选择 → 返回主流程
  
  若所需指标不存在 → 系统推断指标逻辑并生成文字描述 → 用户确认创建 → 自动保存 → 返回主流程
  
  若意图清晰且指标完备 → 系统结合数据模型和表血缘生成查询语句 → 执行查询获取数据 → 返回数据结果 → 结束
  ```
  
- 智能分析-AI通用问答
  ```
  开始 → 用户输入问题 → 系统识别问题类型 →
  
  若咨询指标逻辑或业务口径 → 检索指标库和知识库 → 返回指标定义、计算口径和业务说明 → 结束
  
  若咨询数据表结构或字段信息 → 检索元数据知识库 → 返回表结构、字段说明和关联关系 → 结束
  
  若进行简单取数查询 → 系统解析意图生成查询语句 → 结合表血缘补全关联路径 → 执行查询获取数据 → 返回数据结果 → 结束
  
  若意图模糊则生成澄清问题 → 用户明确后返回对应分支执行
  ```
  
- 我的订阅
  ```
  开始 → 用户进入我的订阅 → 查看已订阅报告列表 → 选择目标报告 → 用户选择时间范围 →
  
  若仅查看 → 按所选时间范围重新生成报告内容 → 展示报告 → 结束
  
  若进行二次分析 → 继承原报告配置结合所选时间范围 → 用户输入新问题或追问 → 系统生成分析结果或新报告 → 返回用户 → 结束
  ```
